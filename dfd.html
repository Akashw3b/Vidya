A variable is the name given to memory location in a program. For example 

  a=30       b="srikumar"       c=71.22

Variable - Container to store a value

Keywords - Reserved words in Python

Identifiers - /function/variable name

Data Types:

Primarily these are following data types in python:

  1. Integers
  2. Floating point numbers
  3. String
  4. Boolean
  5. None

Python is a fantastic language that automatically identifies the type of data for us. 

           
    a = 71     #Identifies a as class -int \n        
    b = 88.76     #Identifies b as class -float \n        
    name = "srikumar"    #Identifies c as class -str \n

Rules for defining a variable name:

(Also applicable to other identifiers)

  • A variable name can contain alphabets, digits, and underscore.
  • A variable name can only start with an alphabet and underscore.
  • A variable can't start with a digit.
  • No white space is allowed to be used inside a variable name.
    • Examples of few valid variable names, Srikumar, srikumar, one8, _akki, aakash, srikumar_bro, etc.
